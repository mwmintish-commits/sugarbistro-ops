# 🍯 小食糖營運系統 sugarbistro-ops

LINE Bot + Web 管理後台，管理 4 間門市的完整餐飲營運系統。

## 功能
- 📱 LINE Bot 打卡、日結、費用回報
- 👥 員工管理、排班、請假、薪資
- 💰 日結、存款、費用（AI 辨識）
- 📊 損益表、門市比較
- 📋 工作日誌、盤點系統
- 🏆 考核、獎金

## 技術
Next.js 14 + Supabase + LINE Messaging API + Claude AI + Google Vision
